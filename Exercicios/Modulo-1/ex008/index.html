Negrito, Itálico e Semântica.

Negrito / Destaque

Neste parágrafo temos um termo em negrito utilizando a tag <b>. A tag <b> NÃO É semântica.

Neste parágrafo temos um termo em destaque utilizando a tag <strong>. A tag <strong> É semântica.

Vale ressaltar aqui que as tags <> e <> não se diferenciam em sua FORMA, mas sim pela sua SEMÂNTICA. Ambos os textos possuem partes em negrito. Porém, o HTML é uma linguagem semântica, onde o código possui referência ao que se entrega. Isso é muito importante para se diferenciar tags de forma de tags semânicas.

Itálico / Ênfase

Neste parágrafo temos um termo em itálico utilizando a tag <i>. A tag <i> NÃO É semântica

Neste parágraf temos um termo em ênfase utilizando a tag <em>. A tag <em> É semântica.

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>. Podemos utilizar o CSS para alterar a cor do background. Conforme vemos neste exemplo, aplicamos a cor de fundo por meio do código hexadecimal #adff2f. Lembrando que a cor da tag <mark>, por padrão, é amarelo.

Texto Grande e Pequeno

Estamos crianco um texto grande e um texto pequeno neste parágrafo.

Lembrando que a tag <big> funciona, porém, atualmente, está obsoleta, segundo a W3C. Já a tag <small> não está obsoleta e encontra-se pronta para uso.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. A tag <u> não é semântica, porém também sublinha o texto. Neste exemplo, utilizamos a tag <ins>

Texto Sobrescrito

Para inserir textos do tipo: x20 + 3.

Texto Subscrito

Para inserir textos do tipo: H2O.
