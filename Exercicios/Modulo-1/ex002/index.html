Parágrafos e Quebra de Linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo entre o par de tags <p> e </p>.

Lembrando que para quebrar linhas, utilize a tag <br>. Para quebrar mais linhas, não utilize mil vezes a tag <br>. Para isto que existe o "css".

Adicionando alguns símbolos especiais ao site, como:
® (Marca Registrada)
© (Copyright)
™ (Trademark)
O Visual Code Studio completa automaticamente na lista de comandos. Ou no PDF desta aula.
