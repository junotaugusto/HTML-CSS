Listas Ordenadas

Listas ordenadas são aquelas que a ordem faz a diferença.

  1. Acordar
  2. Ligar para João
  3. Tomar Café
  4. Escovar os Dentes
  5. Trabalhar

No HTML5 o </li> é OPCIONAL. Portanto, o elemento de uma lista em HTML pode ficar da seguinte maneira:

        <ol>

            <li>Acordar 

            <li>Ligar para João 

            <li>Tomar Café 

            <li>Escovar os dentes 

            <li>Trabalhar 

        </ol>

Existem algumas formas interessantes de alterar uma lista ordenada. Por exemplo, inserindo o parâmetro "type" dentro da tag <ol>. Confira abaixo:
