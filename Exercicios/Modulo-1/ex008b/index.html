Código-Fonte e Pré-Formatação

Códigos-Fonte são partes de código, comandos como, por exemplo: "O comando document.getElementById('teste') é escrito em linguagem JavaScript.". Este tipo de exibição do código fica confusa, conforme está escrito no exemplo, porque as letras, visualmente, não possuem o mesmo espaçamento, porque a fonta não é monoespaçada, ou seja, ela não reserva o mesmo espaçamento lateral para cada letra.

Para fazer isso, utilizamos a tag <code>. Confira neste exemplo: "O comando document.getElementById('teste') é escrito em linguagem JavaScript.".

Outro exemplo seria escrever um código em Python. Confira no exemplo abaixo:

num = int(input('Digite um número: ')) if num % 2 == 0: print (f'O número {num} é PAR') else: print (f'O número {num} é ÍMPAR') print ('Fim do Programa.')

Podemos perceber, entretanto, que o <code> não escreve o código com a indentação. Por isso, iremos escrever abaixo como ficaria o código utilizando espaços e indentação com o comando ' ' e o comando <br>

num = int(input('Digite um número: '))
if num % 2 == 0:
  print (f'O número {num} é PAR')
else:
  print (f'O número {num} é ÍMPAR')
print ('Fim do Programa.')

É possível também escrever este código Python utilizando a tag <pre>, que é a pré-formatação do texto, como pai de <code>. Confira abaixo. 


    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print (f'O número {num} é PAR')
    else:
        print (f'O número {num} é ÍMPAR')
    print ('Fim do Programa.')

Citações

Como diria o pai de um amigo: O computador é um burro muito rápido. Utilizou-se a tag <q> para citar a frase. Lembrando que o HTML é uma linguagem semântica. Por isso, utilizar a tag de citação ao invés de utilizar diretamente as aspas faz muito sentido. Sendo assim, a partir de agora, utilize a tag <q> para citar a frase de alguém.

Citações Completas

Segundo Mauricio Samy Silva, em seu livro "Fundamentos de HTML5 e CSS3":

O World Wide Web Consortium, ou W3C, é um consórcio internacional com quase 400 membros, entre eles, empresas, órgãos governamentais, e organizações independentes que tem por objetivo definir padrões destinados à criação de conteúdos para a web. O termo "Padrões Web" é usado em linguagem de desenvolvimento web para definir técnicas e práticas de criação de conteúdos web em acordo com as recomendações (ou especificações) do W3C.

Note que a especificação "cite" não cria um link para acesso, porém, o endereço fica na parte de trás do site, e os mecanismos de buscas, o google, vão saber que esta frase está se referindo ao endereço destacado no "cite".
