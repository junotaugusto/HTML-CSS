Extensão Gozilla e Gradiente

Na aula anterior, o professor instalou uma extensão chamada "Colorzilla" na Web Store do Google.

Esta extensão tem por objetivo capturar quaisquer cores que o desenvolvedor selecionar. Como foi uma aula curta, não criamos um arquivo para tal. Paciência.

Testando Gradiente

Para inserir uma cor gradiente, o estilo CSS aplicado é o "background-image". Depois, selecione a opção "linear-gradient()". Assim que isto for feito, escolha a direção para a qual a cor irá ficar gradiente e insira quantas cores quiser.
