A centralização de caixas box level não devem ser feitas pela margin-left até chegar ao centro da página. Se isso for feito, e a tela do navegador do usuário for maior ou menor, ele já não centraliza. O meio, em si, é calculado de acordo com o tamanho da caixa. Para centralizar uma caixa utilize a propriedade margin: auto..

O outline possui seu tamanho que pode extrapolar a margem mas ele não aumenta o tamanho da margem.

Utilizando Shorthands

Podemos configurar o padding, border, margin, outline etc utilizando apenas uma linha de código, comumente chamada de shorthand. Para inserirmos os valores que cada atributo vai receber, colocamos , no sentido horário, começando do topo e indo à direita, base e esquerda, esses valores

Porém, na utilização de shorthands, podemos fazer da seguinte maneira:

Digamos que queremos inserir uma margem em determinada caixa. Se quisermos que cada lado tenha 10px de margem, não precisamos inserir este valor em cada lado da margem. Utilizamos o shorthand:

margin: 10px;

Desta forma, todos os lados terão 10px de tamanho. Mas e se quisermos inserir 10px no top e no bottom e 20px no right e left. Faça desta maneira:

margin: 10px 20px;

Então, quando colocamos quatro valores, é a configuração para cima, direita, baixo e esquerda. Quando colocamos dois valores, é a configuração de cima e embaixo iguais e esquerda e direita iguais. Quando colocamos apenas um valor, é o mesmo valor para cada lado.

Para centralizar a caixa, mas manter as configurações de margem de baixo e de cima, utilize o atributo auto na esquerda e direita do código conforme descrito abaixo:

margin: 10px auto 10px auto;

Display: Block ou Inline

Podemos alterar a propriedade de uma caixa de block para inline ou vice versa. Se inserirmos o atributo display:block para uma caixa inline, ela irá se comportar como um bloco, quebrando a linha, podendo configurar o box sizing, inserindo largura, altura etc.

A mesma coisa acontece se colocarmos o atributo display:inline para uma caixa em bloco. Ela irá se comportar como uma caixa inline, perderá os atributos de altura e largura, quebra de linha etc, porém ficará na mesma linha.
