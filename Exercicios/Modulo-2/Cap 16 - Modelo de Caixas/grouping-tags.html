Grouping Tags ou Tags de Agrupamento

As tags de agrupamento sempre foram, basicamente, duas. A <div> como box-level e a <span> como box-inline

Antigamente, nas outras versões do HTML, era comum utilizar várias divs para separar o site, inserindo em casa uma delas ids com os nomes de "conteúdo", "cabeçalho", "rodapé" etc.

Porém, o HTML5 possui semântica, ou seja, existem tags que servem para fazer justamente este trabalho de separação correta do site, a fim de auxiliar o desenvolvedor na leitura do código e os mecanismos de busca.

Algumas das tags de agrupamento são:

  • <header> - Tag de cabeçalho.
  • <main> - Tag de conteúdo principal.
  • <footer> - Tag de rodapé.
  • <nav> - Tag de barra de navegação.
  • <article> - Tag de conteúdo de artigo.
  • <aside> - Tag de créditos ou alguma coisa menor destacando alguém ou alguma coisa.
