Modelo de Caixas

O Que é o Modelo de Caixas?

O modelo de caixa pode ser aplicável ao HTML e ao CSS. É um conjunto que ajuda muito na organização do site. A teoria é a de que o desenvolvedor possui uma caixa na mão. Em um documento HTML, tudo aquilo que é exibido na tela é exibido na forma de caixa. Praticamente, é o que vemos neste site, nestes parágrafos.

A grande maioria das caixas em HTML podem estar uma dentro da outra. Isso facilita muito a disposição do site. Quando inserimos um elemento dentro do outro, tanto em linguagens de programação, como em HTML, nós chamamos de aninhamento.

Quando criamos uma tag <h1>, por exemplo, tudo o que escrevermos dentro da tag ficará "dentro da caixa de h1". Quando fazemos configurações de largura, altura, cor de fundo, fonte etc do seletor h1, na verdade, estamos configurando a caixa na qual ele se encontra.

As caixas possuem propriedades como width (largura), height (altura), margin (margem), padding (preenchimento), border (borda), outline (contorno). Estes são os principais elementos da anatomia de uma caixa.

Tipos de Caixa

Existem dois tipos de caixas:

1 - Box Level - Uma caixa do tipo Box Level, se nós a criarmos dentro de um parágrafo, por exemplo, ela irá pular para a linha de baixo e vai dezenhar uma caixa nova, ou seja, ele sempre inicia uma linha nova. Outra propriedade de box level é que ele sempre ocupa a largura total do viewport (area branca do navegador).

2 - Inline Level - Uma caixa do tipo inline level irá, se sobrar espaço, desenhar a caixa ao lado do conteúdo sem pular uma linha, exatamente do tamanho da caixa de texto e, quando terminar, ele também não irá quebrar a linha, ou seja, ele não irá ocupar 100% do viewport. aa

Extensão Web Developer

Abrindo o web developer, inspecionando a página, aparece algunas configurações já feitas pelo "user agent". O user agent é o próprio navegador. Todo navegador possui uma configuração básica seguida por todos eles.

Por exemplo, o <h1> é um elemento do tipo display:block. Um elemento display:block é um elemento de box level. Na aula, cujo arquivo é este aqui, veja que a tag <h1> recebeu algumas alterações, utilizando o CSS interno. Desta forma, quando alterados quaisquer elementos antes estabelecidos pelo padrão do navegador, a extensão mostra aquele elemento específico riscado e, em cima, destacado como uma alteração do usuário por meio da tag <style>.

Outras propriedades, como alterar na hora a altura ou largura, alterando o peso da fonte, etc, pode ser feito na hora, para melhor visualização do desenvolvedor. Lembrando que as alterações feitas na extensão não alteram o arquivo original.
