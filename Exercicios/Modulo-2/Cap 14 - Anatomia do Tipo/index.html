Anatomia do Tipo

Fontes

Existem diferenças entre o espaço ocupado pelas letras, dependendo da fonte utilizada. É preciso saber as medidas para aplicar as fontes em determinados lugares.

A letra x é o ponto de partida para a construção de todas as fontes porque existe uma métrica chamada "altura xis" que vai definir a altura base de todas as letras minúsculas.

A altura das letras maiúsculas se chama Altura das Maiúsculas. A altura das letras que vazam acima da altura xis é chamada de ascendente. A altura do que vazou para baixo é chamada de descendente.Somando todas as alturas se chama de corpo.

Serifa é aquele pequeno traço que existe em algumas fontes, nas pontas das letras. Ela gera uma linha imaginária para guiar os nossos olhos através das palavras e das letras.

Hastes são as "traves" das letras, as partes retas que formam as letras. Já uma linha que une duas hastes é chamado de filete, como por exemplo ocorre com a letra A.

A parte arqueada das letras é chamada de arco e o "pézinho" das letras é chamado de esporão. A ponta das letras é chamada de vértice.

Categoria de Fontes

  • Serifadas - Possuem serifas. São recomendadas para a impressão em papel.
  • Sans-Serif - Não possuem serifa. São recomendadas para textos longos e impressão em tela.
  • Monoespaçada - Podem ter ou não serifas. Monoespaçadas são as fontes cujas letras que possuem a mesma largura.
  • Handwriting - Fonte escriptada. São as que simulam a caligrafia humana
  • Fonte Display - Não se baseia nos tamanhos, não tem linha base, não se preocupa em obedecer regras anatômicas, são fontes comemorativas.
